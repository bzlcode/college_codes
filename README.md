# DSA-S3

This folder contains a collection of C programs focused on Data Structures and Algorithms (DSA).Below is an overview of the contents of this folder:

## Contents

### Programs

1. **addpolynmultiply.c**
   - Implements polynomial addition and multiplication.

2. **bfs.c**
   - Demonstrates Breadth-First Search (BFS) algorithm for graph traversal.

3. **binarySearch.c**
   - Implements the Binary Search algorithm for searching in a sorted array.

4. **dfs.c**
   - Demonstrates Depth-First Search (DFS) algorithm for graph traversal.

5. **dict_tree.c**
   - Implements a dictionary using tree data structures.

6. **expression_tree.c**
   - Constructs and evaluates expression trees.

7. **find-and-replace.c**
   - A utility program for finding and replacing text in files.

8. **infix.c**
   - Converts infix expressions to postfix expressions.

9. **priorityQ.c**
   - Implements a priority queue using heap data structure.

10. **sorting.c**
    - Demonstrates various sorting algorithms.

11. **sparse_matrix.c**
    - Implements operations on sparse matrices.

12. **warehouse-project.c**
    - A project simulating warehouse management using data structures.

13. **web-nav-using-linkedlist.c**
    - Simulates web navigation (forward and backward) using linked lists.
